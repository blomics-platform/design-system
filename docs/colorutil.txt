Color utility variables
Color variables (Figmaˇs version of design tokens) store reusable values that can be applied to all kinds of design 
properties including fill and stroke colors. They help save time and effort when managing design systems by acting 
as a ˝s
ingle source of truth˛  while allowing designers to create multiple color modes.
Alpha colors
Variables
Use alpha color variables to manage all white and black opacity fill colors in your designs 
across light and dark modes. For more detail on how these variables are structured and 
how to use them, please refer to our Introduction to variables.
